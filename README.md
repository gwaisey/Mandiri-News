# 🏦 Mandiri News App - Mobile Apps Developer Internship
Final Task for Bank Mandiri x Rakamin Academy Project-Based Internship.
**Score: 93/100** 🏆

## 📱 About the Project
Mandiri News is a sleek and modern news aggregator built with **Kotlin**. This app provides real-time news updates with a focus on seamless navigation and professional branding, inspired by Bank Mandiri's visual identity.

## ✨ Key Features
* **Dynamic News List**: Real-time articles using News API.
* **Smart Search**: Auto-scrolls to top when searching to ensure users see the latest results.
* **Interactive Branding**: Clickable Bank Mandiri logo that redirects to the official website.
* **Modern UI/UX**: 
    * **Typography**: Using **Archivo SemiBold** for a premium feel.
    * **High Contrast**: Optimized white-text-on-dark-cards for better readability.
    * **Accessibility**: Proper touch targets (48dp) and content descriptions.

## 🛠️ Tech Stack
* **Language**: Kotlin
* **UI Framework**: XML (RecyclerView, ViewPager2, Glide)
* **Networking**: Retrofit2 & News API
* **Architecture**: MVVM / Clean Architecture (Optional, adjust to yours)
